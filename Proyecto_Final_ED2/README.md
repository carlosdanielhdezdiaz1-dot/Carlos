# Sistema de Gestión de Proyectos

## Estructura de Datos II - Proyecto Final Integrador

### Descripción
Sistema que modela un proyecto mediante tareas organizadas en un Árbol Binario de Búsqueda (ABB) y dependencias gestionadas con un Grafo Dirigido Acíclico (DAG).

### Características
- ABB de tareas ordenado por ID
- Grafo dirigido para dependencias entre tareas
- Detección de ciclos
- Ordenamiento topológico
- Menú interactivo por consola

### Tecnologías
- Java 11+
- Sin dependencias externas
